# vagrant-lamp

This repository create virtual development environment.

## middleware

| package | version |
| ------- | ------- |
| apache  | 2.4.6   |
| php     | 7.0.33  |
| mariadb | 5.5.60  |


## installation

You need next tools:

- [Vagrant](https://www.vagrantup.com/downloads.html)
- [Virtualbox](https://www.virtualbox.org/wiki/Downloads)
- [Ansible](https://www.ansible.com/)

## deploy

```sh
$ vagrant up
```
